# Обновление

Служба базы данных Alauda для MySQL-PXC будет выполнять обновления на основе заданной стратегии обновления:

- **Автоматическое**: Автообновления запускаются немедленно при обнаружении новых версий компонентов.
- **Ручное**: Требуется ручное одобрение перед началом процесса обновления.
