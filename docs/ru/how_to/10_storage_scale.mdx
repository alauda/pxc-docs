# Онлайн расширение тома

## Случай использования

По мере увеличения объема данных важно своевременно расширять хранилище, когда использование приближается к квоте хранилища, чтобы обеспечить непрерывную доступность бизнеса. Процесс онлайн-расширения не повлияет на нормальное предоставление услуг.

## Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. В левом навигационном меню щелкните на **MySQL-PXC**.

    2. Щелкните на ***Имя пространства имен***.

    3. Щелкните на ***Имя экземпляра***.

    4. На вкладке **Детали** нажмите кнопку с правой стороны от запроса на постоянный том.

    5. Нажмите **Расширить** или выберите несколько запросов на постоянный том и нажмите **Массовое расширение**.

    6. Укажите желаемую емкость для расширения.

    7. Нажмите **Расширить**.

       Подождите немного, и вы увидите соответствующее увеличение **Ёмкости хранения** в списке запросов на постоянный том.
  </Tab>
</Tabs>
