# Переключение режима чтения-записи

## Применение

Эта функция поддерживает установку конкретного узла MySQL в режим чтения-записи. Пожалуйста, оцените влияние операции переключения на ваш бизнес и, если необходимо, выполните операцию вне пиковых часов работы.

## Процедура

<Tabs>
  <Tab label="Web Console">
    1. Нажмите на **MySQL-PXC** в левой панели навигации.

    2. Нажмите на ***имя пространства имен***.

    3. Нажмите на ***имя экземпляра***.

    4. На вкладке **Топология** прокрутите вниз, чтобы найти узел MySQL.

    5. Нажмите **Установить в режим чтения-записи** и подтвердите.
  </Tab>
</Tabs>
