# Анализ запросов

Анализируя действия SQL-запросов, вы можете точно определить SQL-операторы, которые влияют на производительность базы данных, или постоянно отслеживать действия запросов для корректировок и оптимизаций операторов.

## Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Нажмите на **MySQL-PXC** в левой боковой панели.

    2. Нажмите на ***Имя пространства имен***.

    3. Нажмите на ***Имя экземпляра***.

    4. На вкладке **Анализ запросов** установите критерии поиска.

       - Чтобы найти операторы, выполненные в определенный временной интервал: выберите временной интервал в разделе **Временной диапазон**.

       - Чтобы найти конкретные операторы: введите ключевое слово(а) в разделе **Поиск по SQL-оператору**, затем нажмите поиск.

    5. Подождите некоторое время и просмотрите результаты.
  </Tab>
</Tabs>
