# Управление пользователями

Эта функция предоставляет графическую функцию управления пользователями. Рекомендуется назначить пользователей базы данных соответствующим разработчикам баз данных в зависимости от потребностей в использовании.

## Предварительные требования

Убедитесь, что экземпляр находится в состоянии **Работает**.

## Создание пользователя

Создайте пользователя и настройте разрешения для этого пользователя. Платформа одновременно сгенерирует учетные данные для аутентификации пользователя для ресурсов Kubernetes - **Секретный словарь**. Для приложений, использующих MySQL, имя пользователя и пароль MySQL могут быть запрошены в Pods через Секретный словарь. Секретный словарь автоматически синхронизирует информацию о пользователе; не изменяйте настройки внутри него.

<Tabs>
  <Tab label="Веб-консоль">
    #### Создание пользователя

    1. Нажмите **MySQL-PXC** в левой навигационной панели.

    2. Нажмите ***Имя пространства имен***.

    3. Нажмите ***Имя экземпляра***.

    4. На вкладке **Управление пользователями** нажмите **Создать пользователя**.

    5. Установите информацию о пользователе.

       **Примечание**: Пожалуйста, надежно храните свой пароль. Чтобы снизить риск утечки пароля, рекомендуется регулярно обновлять пароль.

    6. Чтобы убедиться, что пользователь может нормально работать, пожалуйста, своевременно назначьте пользователю разрешения.

    7. Нажмите **Создать**.

    #### Добавить Секретный словарь (для пользователей, созданных не через платформу)

    Если пользователь базы данных был создан не через платформу, пожалуйста, вручную создайте соответствующий Секретный словарь для них. **Секретный словарь** - это учетные данные для аутентификации пользователя, используемые ресурсами Kubernetes. Если приложению необходимо использовать MySQL, имя пользователя и пароль MySQL могут быть запрошены в Pods через Секретный словарь.

    1. Нажмите **MySQL-PXC** в левой навигационной панели.

    2. Нажмите ***Имя пространства имен***.

    3. Нажмите ***Имя экземпляра***.

    4. На вкладке **Управление пользователями** нажмите ***Имя пользователя***.

    5. Введите пароль, заданный во время создания пользователя, затем нажмите **Добавить**.

    #### Обновить пользователя

    1. Нажмите **MySQL-PXC** в левой навигационной панели.

    2. Нажмите ***Имя пространства имен***.

    3. Нажмите ***Имя экземпляра***.

    4. Перейдите на вкладку **Управление пользователями**.

       - Чтобы обновить пароль: нажмите **Обновить пароль** рядом с пользователем.

       - Чтобы обновить разрешения пользователя (для не-**root** пользователей): нажмите ***Имя пользователя***, затем в деталях пользователя нажмите **Обновить разрешения**.

    5. Измените настройки.

    6. Нажмите **Обновить**.
  </Tab>
</Tabs>
