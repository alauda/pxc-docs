# 发布说明

## v4.0.0

### 新增与优化功能

#### 组件与平台解耦

Alauda 数据库服务的 MySQL-PXC 生命周期已与平台的生命周期解耦，这意味着当客户需要某个功能或修复错误时，MySQL-PXC 团队可以立即发布，而无需等待平台的共同发布。

#### 组件详情页面的实时日志

MySQL-PXC 详情页面的日志已被替换为实时日志，能够即时更新，方便故障排除和问题分析。
