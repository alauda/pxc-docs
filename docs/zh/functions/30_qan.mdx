# 查询分析

通过分析 SQL 查询活动，您可以准确识别影响数据库性能的 SQL 语句，或持续监控查询活动以进行语句调整和优化。

## 操作步骤

<Tabs>
  <Tab label="Web Console">
    1. 点击左侧导航栏中的 **MySQL-PXC**。

    2. 点击 ***命名空间名称***。

    3. 点击 ***实例名称***。

    4. 在 **查询分析** 页签中，设置搜索条件。

       - 要查找特定时间段内执行的语句：在 **时间范围** 部分选择一个时间段。

       - 要查找特定语句：在 **按 SQL 语句搜索** 部分输入关键字，然后单击搜索。

    5. 稍等片刻，查看结果。
  </Tab>
</Tabs>
