# 查询分析

## 功能介绍
通过分析 SQL 查询活动，精准定位影响数据库运行效率的 SQL 语句，或持续监控查询活动并进行语句调整及优化。

## 操作步骤
<Tabs>
<Tab label="界面操作">

1. 在左侧导航栏中，单击 **MySQL-PXC**。

2. 单击 ***命名空间的名称***。

2. 单击 ***实例名称*** 。
    
3. 在 **查询分析** 页签中，设置查找条件。

    * 查找特定时间段内执行的语句：在 **时间范围** 处，选择一个时段。
    
    * 查找特定语句：在 **按 SQL 语句搜索** 处，输入语句关键字，单击查找。    

4.  稍等片刻，并查看结果。    
</Tab>
</Tabs>
