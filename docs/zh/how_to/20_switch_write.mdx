# 切换读写模式

## 使用场景

支持将某个 MySQL 节点设置为读写模式。请评估切换操作对您的业务的影响，必要时可在业务高峰期之外进行操作。

## 操作步骤

<Tabs>
  <Tab label="Web 控制台">
    1. 在左侧导航栏中，单击 **MySQL-PXC**。

    2. 单击 ***命名空间的名称***。

    3. 单击 ***实例名称***。

    4. 在 **拓扑** 页签下，找到 MySQL 节点。

    5. 单击 **设置为读写**，并确认。
  </Tab>
</Tabs>
