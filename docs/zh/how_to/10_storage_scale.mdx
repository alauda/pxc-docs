# 在线卷扩容

## 使用场景
随着数据量的增加，当存储使用量接近存储配额时，及时扩容存储是至关重要的，以确保业务的持续可用性。在线扩容过程不会影响正常的服务提供。

## 操作步骤
<Tabs>
  <Tab label="Web 控制台">
    1. 在左侧导航栏中，单击 **MySQL-PXC**。

    2. 单击 ***命名空间名称***。

    3. 单击 ***实例名称***。

    4. 在 **详细信息** 页签下，单击持久卷声明右侧的按钮。

    5. 单击 **扩容**，或选中多个持久卷声明并单击 **批量扩容**。

    6. 填写希望扩容到的容量。

    7. 单击 **扩容**。

       稍等片刻，您会在持久卷声明列表中看到 **存储容量** 相应增加。
  </Tab>
</Tabs>
