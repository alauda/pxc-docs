# 在线扩容

## 使用场景
随着数据量增长，在存储使用量接近存储配额前应及时扩容，以确保业务的持续可用。在线扩容过程不会影响正常提供服务。

## 操作步骤
1. 在左侧导航栏中，单击 **MySQL-PXC**。

2. 单击 ***命名空间的名称***。

2. 单击 ***实例名称*** 。

3. 在 **详细信息** 页签下，单击持久卷声明右侧按钮。

4. 单击 **扩容**，或选中多个持久卷声明并单击 **批量扩容**。

6. 填写希望扩增到的容量。

5. 单击 **扩容**。

    稍等片刻，可看到持久卷声明列表中的 **存储容量** 对应增加。
